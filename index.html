What this is:

This web application calculates the value of a given Twitter username based on a couple arbitrary metrics. Using the Twitter API, it searches the inputted user's timeline to find the last 20 original tweets out of 200. For each retweet, that's $8; for each favorite, that's $.20; for each follower, that's $2.50.

The graph displays both the daily value (what the day's total tweet values were), and the cumulative total (the user's total value on day 'x').

How to use:

1.) Add in your desired Twitter username.

2.) Click 'Equitize!'

3.) Results should populate if the inputted user has made at least one original tweet.
